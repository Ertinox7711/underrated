LA MÉTHODE

L'élève apprend COMMENT faire. Il utilise les outils lui-même, guidé par les modules + les templates fournis.

01
Construire ton histoire virale
~45 min · Comment créer un univers qui accroche
ConceptBible
  • Les 10 leviers de viralité — pourquoi Skibidi Fruits marche et comment les reproduire
  • Choisir son genre (survie, romance, thriller, action, fantasy) + son univers visuel
  • Créer ses personnages : nom + design + rôle narratif (méthode Fraisita/Cerisa)
  • Construire sa bible de série : 20+ arcs, couples, alliances — le carburant de 60 épisodes
  • Développer son vocabulaire maison — la langue privée qui crée l'appartenance
02
Écrire un épisode qui retient
~60 min · Structure obligatoire + prompts ChatGPT
ScriptPrompts
  • La structure 5 temps imposée : accroche → mise en place → montée → retournement → cliffhanger
  • Les prompts ChatGPT/Gemini exacts à copier pour générer un script en 10 min
  • Calibrer la durée : 60-90 secondes = magie algorithmique (comment compter)
  • Piloter le scénario par les commentaires : lire, filtrer, intégrer en 30 min
  • Les mots déclencheurs dans les titres : Révélations, Confrontations, Départ, Votes
03
Produire l'épisode avec les outils IA
~90 min · Pipeline ElevenLabs → Midjourney → Runway → CapCut
ProductionIA
  • ElevenLabs : configurer sa voix, importer le script, exporter l'audio
  • Midjourney : prompts de personnages cohérents entre épisodes (seed + style lock)
  • Runway Gen-3 / Kling : transformer les images en clips vidéo 4 secondes
  • CapCut : assemblage, coupes <2s, stickers TikTok, sous-titres, export vertical 9:16
  • Vignette : logo 3D doré + titre visible frame 1 — template fourni
04
Distribuer + Monétiser
~45 min · Algo TikTok, brand deals, formation
AlgoRevenue
  • Cycle publication quotidien : 6h annonce → 18h épisode → 22h relance
  • Gérer le manque intentionnel — comment créer l'attente (levier 3)
  • Brand deals : deck, pricing, comment contacter (modèle Betclic/Free/Oasis)
  • Construire et vendre sa propre formation avec la méthode Underrated

ACCOMPAGNEMENT
LES BOTS

L'élève a accès à des bots qui font le travail technique à sa place. Il se concentre sur la créativité et la stratégie, pas sur la technique.

✍️
Bot #1 — Générateur de Scripts
Core
L'élève remplit un formulaire simple (contexte épisode + demandes commentaires). Le bot génère un script structuré complet en 30 secondes, respectant la structure 5 temps Underrated.
  • 1 Élève saisit : numéro épisode + ce qui s'est passé avant + demandes commentaires
  • 2 Bot appelle GPT-4o avec le prompt système Underrated (structure imposée, vocabulaire maison, cliffhanger obligatoire)
  • 3 Script livré en 30 secondes — format séquencé [0-3s], [3-8s], [8-25s]...
  • 4 Version voix-off séparée + suggestions de titres avec mots déclencheurs
  • 5 Élève copie la voix-off dans ElevenLabs → continue son workflow
GPT-4o API Interface web ~30 secondes ~0.05€/script
💬
Bot #2 — Analyse Commentaires TikTok
Levier 8
L'élève colle l'URL de sa vidéo TikTok. Le bot scrape tous les commentaires, les classe par intention et sort un résumé des demandes narratives les plus populaires à intégrer dans le prochain épisode.
  • 1 Élève colle l'URL de sa vidéo TikTok publiée
  • 2 Bot scrape les commentaires (via API TikTok ou scraper)
  • 3 GPT-4 classifie chaque commentaire : demande narrative / vote personnage / hater / marque
  • 4 Résumé livré : top 5 demandes + personnages populaires + score engagement
  • 5 Élève copie les demandes dans le Bot#1 pour générer le prochain script
TikTok scraper GPT-4 classification ~2-3 min Pipeline Bot#2 → Bot#1
⚠️
Ce que les bots NE font PAS dans l'accompagnement : génération voix, images, vidéo. L'élève fait ça lui-même avec ElevenLabs/Midjourney/Runway (appris dans la formation). Les bots accélèrent uniquement les étapes les plus chronophages : script et veille commentaires.
