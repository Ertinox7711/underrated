Audit visuel
IDENTITÉ VISUELLE
Standards obligatoires pour consistance algorithmique.
STYLE PIXAR 3D
Obligatoire — pas de 2D, pas de réaliste, pas de cartoon plat.
Pixar 3D = reconnaissance en 0.3s dans le feed. Le cerveau identifie avant de lire le texte.
Corps humanoïde + tête fruit = identité immédiate. Expressions exagérées (yeux larges, émotions amplifiées).
PALETTE SATURÉE
1 couleur = 1 personnage — invariable entre tous les épisodes.
FRAISITA — #DC143C
FRAISIO — #1E64FF
CERISA — #9400D3
BANANELLO — #FFD700
DESIGN PERSO
Corps : humanoïde stylisé, tête grande, corps court.
Tête : fruit 3D Pixar — texture réaliste stylisée.
Expressions : exagérées, lisibles en miniature. Sourcils épais, bouche grande.
Tenue : fixe par personnage — identité = reconnaissance immédiate.
Tool : Nano Banana → paramètres figés par character sheet.
DÉCORS & AMBIANCE
Arène : inspirée Squid Game — formes géométriques, couleurs tranchées, ombres dures.
Contraste : fond sombre si perso clair. Rim light sur personnages.
Tools : Midjourney V7 (décors) + Runway (mouvement caméra).
POURQUOI LE PIXAR MARCHE SUR L'ALGO
Reconnaissance
Le cerveau identifie "Underrated" en 0.3s — avant le son, avant le texte.
Rétention
Pixar 3D = qualité perçue élevée → l'algo TikTok booste les vidéos "production value".
Partage
Style unique = davantage partagé. "T'as vu les fruits ?" = branded recall naturel.
Réwatch
Attrait universel enfants → adultes. Regardent une 2e fois par plaisir visuel pur.
Format validé
STRUCTURE ÉPISODE
60–90 secondes. Chaque phase a un rôle algorithmique précis.
Timing Phase Objectif Levier algo
0–3sACCROCHEExtrait moment le plus intense de l'épisodeStop scroll — 0.3s threshold
3–8sINTROLogo animé + "Épisode X — Underrated Game"Brand recall + curiosité résiduelle
8–25sSETUPRécap jeu ou annonce enjeu — contexte rapideWatch time qualifié — 25%
25–50sACTIONDéroulement — coupes <2s, dialogues Gen ZRétention >75% — signal boost
50–55sCHOCÉlimination inattendue ou révélationRéwatch trigger
55–60sCTACliffhanger + question commentaire ouverteComments + share + follow
Règle montage
COUPES <2S
Chaque plan >2s = risque sortie. Couper avant que le cerveau s'ennuie. Exception : silence avant élimination (3-4s max).
Dialogues
GEN Z ONLY
Courtes phrases. "bro", "fr fr", "no way", "c'est over". Émotion directe. Écrits pour être lus (sous-titres) autant qu'entendus.
Cliffhanger
SANS RÉSOLUTION
Jamais résolu dans l'épisode. La réponse = raison de revenir demain 18h. Question ouverte : "Qui va survivre ?"
Mécaniques algo
LEVIERS RÉTENTION
Ce qui fait rejouer, commenter et s'abonner — à intégrer dans chaque épisode.
Re-Watch Triggers
PARADOXE VISUEL
Fruits qui jouent à des jeux de survie = dissonance cognitive. Le cerveau ne sait pas catégoriser → rejouage pour "comprendre".
ESTHÉTIQUE ABRUTISSANTE
Pixar hypersaturé, trop beau pour être ignoré. Qualité visuelle = le spectateur regarde une 2e fois pour le plaisir des yeux.
RÉVÉLATION CACHÉE
Indices visuels dans le background que seul le 2e visionnage révèle. "Attend, Fraisio regardait déjà le couteau à 0:32..."
Comment Triggers
VOTE & INJUSTICE
"C'est injuste !" = commentaire garanti. L'élimination d'un personnage aimé déclenche la réaction émotionnelle.
QUESTION OUVERTE
CTA final : "Qui devrait être éliminé ?" / "Qui survivra la semaine prochaine ?" — Question = raison de commenter.
SCÉNARIO PAR COMMENT
"Je lis vos commentaires pour écrire l'épisode 12." → investissement émotionnel → partage organique.
VOCABULAIRE MAISON
Inventer des termes propres. "Les Fraiseurs" / "Game Over Bro" = culture propriétaire. Communauté = langage commun.
Subscribe Triggers
CLIFFHANGER CALIBRÉ
Fin d'épisode sans résolution. La réponse vient demain 18h. S'abonner = ne pas rater la suite.
DROP QUOTIDIEN
Daily 18h. Le cerveau apprend le rituel → habitude. S'abonner = notification = rituel maintenu.
CULTURE DES COUPLES
Ship Fraisita × Fraisio = attache émotionnelle. La communauté défend "son couple" → follow pour voir l'évolution.
