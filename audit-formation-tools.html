ARCHITECTURE
DE LA FORMATION

La méthode Underrated en 4 modules. Chaque module ancre un ou plusieurs leviers de viralité.

01
Trouver le concept viral
~45 min · Leviers 1, 4, 5
CONCEPT Fondations
  • Pourquoi le format Squid Game/Fruit Game marche : analyse des 10 leviers de viralité
  • Choisir son genre (survie, romance, thriller, action, fantasy) et son univers visuel
  • Créer ses personnages : nom + suffixe italianisant + design cohérent (ex: Fraisita, Cerisa)
  • Construire la bible de la série : 20+ arcs narratifs, couples, alliances (→ levier 4)
  • Développer son vocabulaire maison — langue privée qui crée l'appartenance (→ levier 5)
  • Valider son concept avec ChatGPT avant de tourner la moindre seconde
02
Pipeline IA : Script → Voix → Vidéo → Montage
~90 min · Leviers 2, 3, 6
PRODUCTION Core
  • Structure obligatoire de chaque épisode : 5 temps (accroche → mise en place → montée → retournement → cliffhanger)
  • Prompt engineering ChatGPT/Gemini pour générer scripts en 10 min max
  • Génération voix-off : ElevenLabs / clonage vocal — ton, débit, breaks dramatiques
  • Génération images/vidéos personnages : Midjourney + Runway / Kling — cohérence visuelle entre épisodes
  • Montage CapCut : coupes <2s (→ levier 2), stickers TikTok, voix-off 4e mur
  • Vignette : logo 3D doré + titre visible frame 1 — jamais d'ouverture à froid (→ levier 1)
  • Durée cible : 60-90 secondes (→ levier 10 : magie algorithmique)
  • Workflow complet en 7h/épisode, coût : quelques dizaines €
03
Stratégie TikTok + Algorithme
~60 min · Leviers 3, 7, 8, 10
ALGO Distribution
  • Cycle de publication quotidien : 6h annonce → 18h épisode → 22h re-visionnage → matin reposts
  • Gérer le manque intentionnel : épisode absent = 10k+ commentaires "où est l'épisode ?"
  • Piloter le scénario par les commentaires en temps réel (→ levier 8) : ChatGPT réécrit en 30 min
  • Exploiter l'engagement négatif : haters = vendeurs gratuits (→ levier 7)
  • Construire les 4 couples = 20 combinaisons d'alliances = identité tribale (→ levier 4)
  • Cross-posting : X / Instagram / YouTube Shorts — amplification hors TikTok
  • Analytics : ratio partages/vues cible >2% (291k partages épisode 1 = 2.4% = exceptionnel)
  • Texte déclencheur dans les titres : Révélations, Confrontations, Départ, Votes
04
Monétisation : Brand Deals + Formation
~60 min · Levier 9
REVENUE Business
  • Brand deals : comment contacter, deck de présentation, pricing (30-70k€ deal majeur)
  • Intégration marque organique dans l'épisode — l'audience doit trouver ça cool (Betclic: 28k likes, Free: 91k likes)
  • Construire la formation : de la méthode au Teachable en <30 jours
  • Tunnel de vente : TikTok bio → landing page → Teachable 99€ → accompagnement 1200€
  • Séquence email Beehiiv : bienvenue → valeur → offre → relance
  • TikTok Creator Fund : secondaire (0.05-0.10€/1000 vues) — ne pas compter dessus
  • Scale : de 30 ventes/mois (3k€) à deal Oasis type canettes édition limitée
Référentiel

SPECS BOTS

6 bots à développer. Chaque bot automatise un levier de viralité. Specs techniques pour l'expert.

💬
Bot #1 — Comment Scraper & Analyzer
P1 CRITIQUE
Objectif

Scraper les commentaires TikTok en temps réel, les classifier par intention (demande narrative, vote personnage, critique, hater) et sortir un résumé actionnable pour piloter le script du prochain épisode. Active le levier 8.

Specs techniques
SourceTikTok API unofficial (scraper) ou TikTok Research API
TriggerCron toutes les 15 min après publication
ClassificationGPT-4 API — catégoriser + scorer chaque commentaire
OutputDashboard web + notification Discord + résumé JSON
StackPython + Playwright/Requests + OpenAI API + SQLite
Features
  • Top 10 demandes narratives les plus likées
  • Vote popularity par personnage (Fraisita vs Cerisa vs...)
  • Détection commentaires de marques (signal brand deal organique)
  • Score "hater ratio" — signal que la vidéo buzze pour les bonnes raisons
  • Alerte si commentaire dépasse 5k likes → génère suggestion de scénario
Levier 8 Python GPT-4 TikTok
✍️
Bot #2 — Script Generator
P1 CRITIQUE
Objectif

Générer automatiquement le script du prochain épisode en fonction des commentaires scrappés + de la bible de série. Produit script en 5 minutes, structuré en 5 temps obligatoires avec cliffhanger. Active les leviers 6 et 8.

Specs techniques
InputJSON commentaires Bot#1 + bible série + dernier épisode
ModèleGPT-4o avec prompt système fixe (structure 5 temps)
OutputScript TXT + version voix-off séparée + tags CapCut
Durée cible60-90 sec (validé par comptage mots — 130-180 mots/min)
StackPython + OpenAI API + interface web simple
Prompt système (template)
  • Structure imposée : accroche → mise en place → montée → retournement → cliffhanger
  • Vocabulaire maison obligatoire (noms des personnages, expressions série)
  • Intégration de minimum 1 demande narrative du top commentaires
  • Fin = question ouverte OU révélation incomplète — jamais de résolution totale
  • Ton : voix-off qui casse le 4e mur, adresse l'audience directement
Levier 6 Levier 8 GPT-4o Python
📅
Bot #3 — Auto Publisher
P1 CRITIQUE
Objectif

Publier automatiquement l'épisode à 18h chaque jour sur TikTok + YouTube Shorts + Instagram Reels + X. Maintient le rituel de publication quotidienne sans intervention manuelle. Active le levier 3.

Specs techniques
TriggerCron 18h00 — si fichier vidéo présent dans dossier /queue
PlateformesTikTok API + YouTube Data API + Meta Graph API + X API
CaptionAuto-générée par Bot#2 — hashtags + appel à l'action
FallbackAlerte Discord si publication échoue (file manquante, API down)
StackPython + APIs officielles + n8n pour orchestration
Workflow
  • 6h : bot post annonce "Episode X aujourd'hui à 18h" sur TikTok (levier 3)
  • 18h : bot upload vidéo + caption générée automatiquement
  • 18h05 : bot notifie Discord #publication avec lien + stats initiales
  • 22h : bot re-poste teaser épisode suivant si vues > seuil (ex: 10k)
Levier 3 TikTok API n8n Cron
📊
Bot #4 — Analytics Tracker
P2 IMPORTANT
Objectif

Tracker les métriques clés de chaque épisode en temps réel. Calculer automatiquement le ratio partages/vues (cible: >2%), détecter les signaux de viralité et alerter l'équipe. Dashboard accessible depuis l'app Underrated.

Specs techniques
SourcesTikTok API + YouTube API + Instagram API + X API
MétriquesVues, likes, shares, comments, saves, rétention, reach
KPI cléRatio partages/vues — alerte si >2% (signal viral)
StockageFirebase Firestore — accessible dans app war room
FréquenceToutes les heures H+1 à H+24 après publication
Alertes automatiques
  • 🟢 Viral signal : ratio partages >2% dans les 3 premières heures
  • 🟡 Watch signal : rétention <70% — flag pour ajuster format
  • 🔴 Drop alert : vues plateau après 6h — envisager annonce épisode suivant
  • 💰 Brand signal : commentaire de compte marque vérifié détecté
Analytics Firebase Multi-API
💼
Bot #5 — Brand Deal Radar
P2 IMPORTANT
Objectif

Détecter automatiquement les marques qui commentent/likent les vidéos (signal intérêt organique), scorer leur potentiel deal, générer un email d'outreach personnalisé et l'envoyer pour review à Mathieu.

Specs techniques
SourceBot#1 (commentaires) + follower scan (comptes vérifiés)
ScoringGPT-4 — audience fit, budget estimé, historique deals IA
OutputDraft email outreach + fiche marque + canal contact
ApprovalDiscord DM Mathieu — "Approve/Edit/Skip" avant envoi
CRMLog dans Notion CRM automatiquement
Email outreach template (généré par IA)
  • Stats clés série (vues total, audience demo, engagement rate)
  • Preuve sociale : mentions de la marque dans les commentaires (si applicable)
  • Proposition d'intégration native dans un épisode (pas pub classique)
  • Pricing suggéré par IA basé sur benchmarks (Betclic/Free type)
Levier 9 GPT-4 Notion Discord
🎓
Bot #6 — Formation Funnel Bot
P3 SCALE
Objectif

Automatiser le funnel de vente formation. Qualifier les leads entrants (DM TikTok, email Beehiiv), segmenter 99€ vs 1200€, déclencher la séquence email adaptée, notifier Ilame pour closing 1200€.

Specs techniques
EntréesBeehiiv webhook + TikTok DM scraper (si API dispo)
QualificationQuestionnaire 3 questions → scoring → segmentation auto
SéquencesBeehiiv API — 5 emails J0/J1/J3/J5/J7 selon segment
Closing signalDiscord alerte Ilame si lead 1200€ clique lien mais n'achète pas
Stackn8n + Beehiiv API + Stripe webhooks + Firebase
Séquence email 5 jours (schéma)
  • J0 : Bienvenue + accès ressource gratuite (extrait méthode)
  • J1 : Histoire Underrated + preuve sociale (vues/revenus)
  • J3 : Le pipeline IA en détail — valeur pure avant vente
  • J5 : Offre 99€ — urgence douce (places limitées coaching)
  • J7 : Dernière chance + témoignage + lien direct Teachable
Funnel n8n Beehiiv Stripe
