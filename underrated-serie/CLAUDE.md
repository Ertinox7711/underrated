# Underrated — Série IA Virale

Dossier dédié à la production de la série : scripts, visuels, personnages, épisodes.

## Concept
Série IA quotidienne style Squid Game. Personnages fruits. Jeux de survie + éliminations.
Cible : TikTok / YouTube Shorts. Format 60-90 secondes. Drop daily 18h.

## Personnages confirmés
- **Fraisita** — féminine, fraise, rouge, meneuse
- **Fraisio** — masculin, fraise, bleu, stratège
- **Cerisa** — féminine, cerise, cramoisi, imprévisible
- **Bananello** — masculin, banane, jaune, naif attachant
- Personnages 5 à 8 — à définir

## Stack production
- Scripts : ChatGPT Pro
- Voix : ElevenLabs Pro
- Vidéo : Runway / Veo 3
- Visuels : Midjourney V7
- Montage : CapCut Pro

## Structure épisode (60-90s)
- 0-3s : accroche (extrait moment intense)
- 3-8s : logo + titre "Épisode X — Underrated Game"
- 8-25s : récap ou annonce jeu
- 25-50s : déroulement (coupes <2s, dialogues Gen Z)
- 50-55s : élimination ou révélation choc
- 55-60s : cliffhanger + CTA commentaire
