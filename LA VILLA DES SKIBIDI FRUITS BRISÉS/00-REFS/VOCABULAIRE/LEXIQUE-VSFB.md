## 🔥 ARGOT SEXUEL (à utiliser avec subtilité dans les dialogues)

| Mot | Définition | Exemple dans VSFB |
|-----|-----------|-------------------|
| **doro / adorer** | Avoir des rapports sexuels | "Il a rien fais le doro avec elle ?" |
| **ken** | Avoir des rapports sexuels (verlan de "niquer") | "Ils ont kéné ou pas ?" |
| **graille / grailler** | Manger — mais aussi argot pour rapport sexuel selon contexte | "Y'avait de la graille hier soir ou pas ?" |
| **s'envoyer** | Avoir des rapports sexuels | "Il s'est envoyé Uvalia c'est clair" |
| **défouriller** | Avoir des rapports sexuels | rare — usage comique uniquement |
| **collé(e)** | Très proche, intime physiquement | "Ils étaient vachement collés hier soir..." |
| **paf** | Pénis (argot direct) | à éviter sauf pour gag Banito |
| **boulangerie est ouverte** | Métaphore : les aventures commencent | "Messieurs, la boulangerie est ouverte" |
| **coquine** | Malicieuse, séductrice | "Pechita joue la coquine mais c'est calculé" |

## 📝 NOTES POUR LES DIALOGUES VSFB

### Intensité selon personnage :
- **NARITA** 🍊 — Parle avec tout le lexique, naturellement, spontané. Gros/frérot/wesh
- **BANITO** 🍌 — Essaie de parler jeune mais se plante, mauvais timing
- **SANDINO** 🍅 — Jamais d'argot en public (trop poli), argot uniquement en aparté caméra
- **UVALIA** 🍇 — Zéro argot. Parle froid et précis. Maximum une fois par épisode "chelou"
- **LIMONO** 🍋 — Argot contrôlé, cool, jamais forcé. "Mondial" "kiffer"
- **PECHITA** 🍑 — Code switch : argot avec Narita, français chic avec Limono
- **FRÉSIO** 🍓 — Argot naturel et positif uniquement. Jamais d'insultes
- **MYRTANA** 🫐 — Silence ou maximal 3 mots. Jamais d'argot

### Mots à bannir (trop vulgaires pour TikTok monétisé) :
- ken / défouriller / paf → remplacer par suggestion / insinuation
- tana → utiliser uniquement en confessionnal
- sa mère → OK oral, éviter les sous-titres

### Mots signature par personnage :
- NARITA : "dinguerie", "malade", "archi", "gros"  
- BANITO : "w...wesh", "chelou" (mal utilisé), "en PLS"
- SANDINO (aparté) : "bolosse", "enfumé", "cramé"
- LIMONO : "mondial", "kiffer", "chaud"
- PECHITA : "seum", "green flag", "pépite"
- FRÉSIO : "malade", "frère", "ouf"
