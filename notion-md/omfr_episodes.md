## E1 — "L'Île de la Skibidi Tentafruit" (12.1M / 1M likes / 12.3K cmts / 291.2K shares / 125.9K saves)
**Hook (3s)** : Logo titre "L'Île de la Skibidi Tentafruit" en doré 3D à la Pixar, sur fond de plage tropicale. Cuts rapides sur les couples de fruits anthropomorphes — immédiatement lisible comme parodie de télé-réalité.
**Premise** : 4 couples-fruits testent leur fidélité sur deux îles paradisiaques (hommes vs tentatrices, femmes vs tentateurs). Reprise stricte du format "L'Île de la Tentation" / Love Island.
**Story arc 60-90s** :
1. Setup : voix-off présente le concept
2. Présentation rapide des couples (cuts \< 2s par perso)
3. Première tension : un fruit regarde un autre fruit
4. Cliffhanger : "ça va déraper"
**Visual language** : Pixar-coded, couleurs saturées (rouges fraise, jaunes banane, bleus myrtille), éclairage golden hour permanent. Personnages humanoides avec tête-fruit.
**Pacing** : \~1.5 cut/sec, dialogue dense, sticker overlays réguliers (style TikTok meme).
**Cliffhanger** : Annonce d'un nouvel arrivant.
**Audio** : Track "Love Island Extended Remix de starz" — hook musical réutilisé sur tous les ép (= identifiant sonore de la série).
**Brand archetype** : Reality-TV satire / parodie soft-NSFW.
**Comments-driven momentum** : 291K shares = la vidéo a explosé en DM-friend-to-friend. Comments montrent rewatch systématique ("qui est revenu après le final").
## E9 — "Le nouveau tentateur" (25.7M / 1.8M likes / 109.6K cmts / 189.2K shares / 194.7K saves) — PIC DE LA SÉRIE
**Hook (3s)** : Plan large d'une nouvelle scène avec arrivée d’un personnage "tentateur" (citron — Citronelo). Le titre “nouveau tentateur” est un hook explicite.
**Premise** : Casting reset — un nouveau perso débloque de nouvelles tensions. C'est exactement le "Bombshell" de Love Island.
**Story arc** :
1. Cold-open : Fraisita lit une lettre de rupture
2. Arrivée du nouveau
3. Réaction des autres fruits (jalousie + curiosité)
4. Scène intime sur une balancelle (= visuel scrappé : Citronelo + Fraisita assis)
5. Cliffhanger : qui va le choisir ?
**Visual language** : Scènes “soft-romance” (jardin tropical, balancelle, nuit, robe blanche). Fraisita dessinée avec collier en croix (codes esthétiques Gen Z catholic-core).
**Pacing** : Plus lent que E1 — pauses sur les regards, gros plans émotionnels.
**Cliffhanger** : Le choix de Fraisita reste ouvert.
**Brand archetype** : Soap-opera ado.
**Pourquoi c'est le pic** : (a) arrivée d'un nouveau perso = reset attentional, (b) escalade émotionnelle bien placée (mid-season), (c) cliffhanger romance ouvre 4-5 ships possibles.
## E12 — "Les Révélations pt.1" (21.9M / 1.7M likes / 73.8K cmts / 203.2K shares / 155.6K saves) — SPONSO OASIS
**Hook (3s)** : Gros plan sur **Fraisita en larmes**, robe échancrée, collier en croix. Plan close-up extrêmement émotionnel. Aucun dialogue sur les 3 premières secondes — juste le visage.
**Premise** : Révélation de trahisons stockées depuis E5-E11. Le titre “Révélations” est un déclencheur émotionnel testé (E5, E6, E19 portent le même mot et perforent toutes).
**Story arc** :
1. Montée d'émotion (Fraisita pleure)
2. Flashback / vidéo cam de surveillance
3. Dialogue cru (“y’a les caméras fais gaffe” — cité par les fans)
4. Réaction des autres fruits
5. Reveal : entrée d’un perso Oasis-coded (Pommita/Poirita)
**Brand integration** : Badge `Collaboration commerciale` officiel + Oasis a sorti des canettes collector apple-pear avec Pommita & Poirita. Selon [dansta.pub](http://dansta.pub) : “musique Frsh d’Oasis en background sur E11, mention officielle E12” — deux épisodes pour faire teaser-reveal.
**Brand archetype** : Drama / révélation à la Real Housewives.
**Insight** : Le top comment est un brand stunt (“La Skibidi canette en édition limitée”). Les marques s’injectent dans les comments avant même de payer un placement.

## Final — "Les Confrontations" (4.2M / 599.9K likes / 36.5K cmts / 76.3K shares / 71.8K saves)
**Hook** : Titre "ÉPISODE FINAL" + "Les Confrontations" — promesse de règlement de comptes.
**Story arc** :
1. Réunion finale
2. Confrontations entre couples
3. **Twist : révélation d'un bébé** (cité par fans : “le bébé il s’appelle multifruit”, “l’enfant de fraisita”)
4. Choix final de Fraisio (qui ne reste pas avec Cerisa = déception fan)
**Pourquoi sous-performance à 4.2M** :
- Déception narrative documentée dans les comments
- Final posté alors que la fatigue de format avait déjà commencé (E20-22 à 5M)
- Manque de paiement émotionnel sur le ship principal
- Le ratio likes/vues reste élevé (14%) = la base hardcore est là, c’est le grand public qui a décroché
**Leçon Underrated** : ne pas étirer une saison au-delà de 15-18 épisodes max. Le pic était E9-E12, ils auraient dû conclure à E15.
## Pattern d'épisodes manquants (4, 10, 13, 14, 21)

Les comments demandent **explicitement l'épisode 10** : “Il est où mon épisode 10 là 😪” — cette frustration *augmente* l'engagement (manque = désir).
