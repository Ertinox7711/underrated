> [!summary]
> **TL;DR**
> - Short-form = même ADN qu'un film : setup / tension / payoff, mais compressé en 30-90s
> - **Beat sheet 60s** : hook (0-5s) → setup (5-15s) → escalation (15-30s) → turn (30-45s) → cliffhanger (55-60s)
> - Cliffhangers + episodic + variable ratio reinforcement = même psy que casino / Netflix binge
> - Bite-sized 30s-2min = sweet spot bingeability
> - **POV format** + visage humain expressif = mirror neurons → stop scroll
> - Daily drop = entraîne une routine d'attente (rendez-vous algorithmique)
## 1. Structure 30/60/90 secondes
### Format 30s (3 actes compressés)
- Setup (0-3s) : Personnage + situation + hook visuel/textuel
- Tension (3-25s) : Problème, conflit, escalade ; questions ouvertes
- Payoff (25-30s) : Résolution OU cliffhanger renvoyant à ep 2
### Format 60s (5 beats classique)
1. **Hook** (0-5s) : phrase d'accroche + question / promesse
2. **Setup** (5-15s) : enjeu, qui, où, pourquoi
3. **Escalation** (15-30s) : montre la complication
4. **Turn / Reveal** (30-45s) : moment de bascule
5. **Cliffhanger** (55-60s) : "épisode 2 demain"
### Format 90s (slow burn / développement personnage)
- 0-5s hook + question
- 5-30s contexte + character
- 30-60s développement + obstacle
- 60-80s climax
- 80-90s payoff partiel + teasing suite
## 2. POV format — mécanique
- POV = "Point Of View" : le viewer est placé dans la scène ("POV: ton boss te dit que...")
- Texte overlay = situateur instantané (remplaçant l'exposition)
- Idea : implique le viewer en 1s = +retention
- Variantes 2025 : POV journal intime / POV interne (voice-over intérieur) / POV split-screen
- Sur TikTok FR, formats POV humour + relations = top engagement chez 16-24
## 3. Reality TV beats appliqués
- **Confessional** : voice-over "je vais te raconter ce qui s'est passé..."
- **Cold open** : on entre au milieu d'un drame, on remonte ensuite
- **Rachat / trahison** : structure répétitive de Love Island, MAFS, etc.
- **Bracket / tournament** : "Série 1 vs série 2, qui gagne ?"
- **Reaction face cam** insert sur visuel = signal authenticité
## 4. Cliffhangers + psy du binge
- **Variable ratio reinforcement** : même mécanisme que slot machines
- Cliffhanger = boucle ouverte que le cerveau veut fermer (Zeigarnik effect)
- 3 types qui fonctionnent :
	1. **Question explicite** ("...mais ce qui s'est passé après va te choquer")
	2. **Information manquante** ("je vous explique demain pourquoi j'ai fait ça")
	3. **Action interrompue** (cut au moment du climax visuel)
- Le "Part 2" est devenu un format : *Drama Update* TikTok atteint 10M+ vues régulièrement
- *Données limitées* sur conversion exacte mais consensus : Part 2 retient 40-60% de l'audience Part 1
## 5. Daily-drop psychology
- Conditionne un **rendez-vous algorithmique** : l'audience attend, donc revient sur le profil
- Créé une **micro-routine** dans le feed (l'app pré-charge votre contenu pour les fans)
- Loop : hook → cliffhanger → attente → payoff jour 2 → hook 2
- Fonctionne mieux avec **horaire fixe** (FR : 18h-21h)
- Burn-out créateur élevé → arsenal de contenus pre-batchés recommandé (10-30 épisodes en avance)
## 6. Archetypes Gen Z 2025-2026
- **The Realist** — "je dis ce que personne dit" (Hugo Décesalle)
- **The Receipts Bringer** — montre les preuves chiffrées (InvestiQuoi)
- **The Underdog** — parti de zero, transparent (beaucoup de créateurs argent)
- **The Insider Whistleblower** — ex-pro qui balance (divers ex-banque, ex-RH)
- **The Comedic Sage** — humour + insight (TheoLaě)
- **The Gentle Mentor** — calme, pédagogue (quelques chaines IA)
- **The Drama Curator** — raconte des histoires d'autres (story-time TikTok)
## 7. AI faceless series — structure type
- Bite-sized **30s-2 min**
- Cast récurrent (même voice + même visuel = familiarité)
- 1 conflit = 1 épisode
- Arc global sur 10-30 épisodes
- Outils 2025 : Sora 2, Veo 3, ElevenLabs, Runway, HeyGen, Krea, Higgsfield
- Format éprouvés : *true crime fictif*, *quiz/trivia visuel*, *mini-drama 60s*, *educational explainer sérialisé*
## 8. Frameworks de scripting
### Framework A — PASCAL
- **P**roblem (3s)
- **A**gitate (5s)
- **S**olution (10s)
- **C**ase study (10s)
- **A**ction (5s)
- **L**oop (cliff back à P)
### Framework B — The 3 Whys
- Why now ? (urgence)
- Why this ? (spécificité)
- Why you ? (autorité)
### Framework C — The Curiosity Gap (Buzzsumo / Backlinko study)
- Promise + Specificity + Twist
- Ex: "3 trucs que personne te dit pour [outcome] — le n°3 marche pas comme tu crois"
