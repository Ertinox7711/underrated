## 1. Cas d'étude : Skibidi Toilet
- Créé par Alexey Gerasimov (DaFuq!?Boom!) avec Source Filmmaker (3D, pas génératif strict mais série surréaliste)
- **65 Md+ vues** cumulées à fin 2023 ; ~17 Md sur la chaîne principale en <2 ans
- **43M abonnés**, ~1 Md vues/mois sur le pic
- Revenu estimé : **$100k → $23M/an** (us.youtubers.me)
- Deal **Invisible Narratives** (Hollywood) pour TV/film/products
- Format : Shorts 15-60s, narrative sérialisée, dialogue minimal, sound design fort
- Take-away : preuve de concept que **short-form narratif faceless peut surpasser MrBeast en views**
## 2. Cas d'étude : Fruit Love Island (mars 2026)
- Créateur : **@ai.cinema021** (identité inconnue)
- Lancement : 13 mars 2026
- 9 jours : **3M+ followers, 300M+ vues** — *fastest TikTok account growth ever*
- Format : parodie Love Island avec fruits humanoides (Strawberina, Bananito, apple host)
- Episode 15 : 39M vues seul
- **Pourquoi viral** : pas l'IA, mais le format — episodic daily, fan voting, plot submissions
- TikTok a retiré des épisodes → migration YouTube + arrêt 28 mars
- Outils utilisés : combo (visuel + voice + script génératifs)
- Take-away : **format > technologie**. Daily drop + interactivité = recette
## 3. État de l'art outils 2026
### Top tools par cas d'usage
- **Veo 3.1** (Google) : T2V + audio natif, API ~$0.10/s, default safe overall
- **Runway Gen-4.5** : T2V/I2V control, $12-76/mois, camera moves prod control
- **Kling 3** : T2V long, ~$10-30/mois, 60s+ shots motion

- **ElevenLabs** : Voice + dubbing, $5-99/mois, voice-over multilingue
- **Sora 2** : T2V (sunset 24/09/2026), $20/mois ChatGPT+, legacy
- **Higgsfield** : Cinematic shots, $9-49/mois, B-roll VFX

## 4. Threshold de qualité — ce qui passe / ce qui déclenche le slop reflex
### Passe (acceptable Gen Z)
- Voice IA bien réglée (ElevenLabs niveau Pro), accent FR cohérent
- Style visuel cohérent across episodes (même look)
- Lip-sync correct si avatar talk
- Captions humaines (écrites, pas auto-laissées)
- Erreurs anatomiques rares ou cachées au cut
### Déclenche "slop" (rejet Gen Z)
- 6 doigts visibles, faces qui morphent
- Voix robotique métallique (basic free TTS)
- Cuts mal sync sur la voix
- Stock-music sur stock-music
- Sujets random sans arc
- *Acceptable si auto-dérisoire, pénalisé si prétend qualité humaine*
## 5. Tolérance audience (données 2025-2026)
- **86%** consommateurs globaux veulent **disclosure AI** (étude 2026, Tribune)
- **77%** veulent savoir si contenu généré IA ; 31% refusent de consommer sans info
- Acceptance par catégorie :
	- Entertainment : **53%**
	- Advertising : **47%**
	- News : **21%** (effondrement)
- Plus haute acceptance si **human oversight** affiché
- *Données Gen Z FR spécifiques limitées*
- Concerns top : misuse 46%, misinfo 42%
### Implication pour Underrated
- Mention disclaimer subtle dans bio + 1ère vidéo de saga
- Format "AI-assisted" plutôt que "AI-only" (même si vrai) = trust +
- Si formation éducative → voix humaine recommandée ou avatar haute-fidélité
- AI parfait pour entertainment / story-mode / hooks visuels
- AI faible pour authority / proof / case studies
## 6. Workflow type 2026 (faceless AI series)
```
1. Concept / arc (humain) — 30 épisodes brief
2. Script (Claude/GPT, prompt structuré) — Beat sheet 5-step
3. Voice (ElevenLabs Voice Lab, voice persistante)
4. Visual generation (Veo 3.1 / Runway Gen-4.5)
5. B-roll génératifs (Higgsfield, Krea)
6. Edit (CapCut Pro)
7. Captions (Submagic auto + cleanup manuel)
8. Thumbnail (Photoshop / Krea / Midjourney)
9. Schedule (Later, Buffer, Postiz)
10. Engage (humain les 30 premières min)
```
### Temps prod par épisode (60s)
- Setup pipeline initial : 2-4 jours
- Par épisode steady-state : **45 min - 2h**
- Coût outil : ~$3-10
- Batch 30 ep en 2 semaines = réaliste solo
## 7. Tendances 2026 à surveiller
