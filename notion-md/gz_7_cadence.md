## 1. Posting frequency — par plateforme
- TikTok : 3-5x/sem min, 1-2x/jour optimal, 3-4x/jour avec batch max
- Reels : 3x/sem min, 5-7x/sem optimal, 1-2x/jour max
- Shorts : 3x/sem min, 1x/jour optimal, 2x/jour max
- YouTube long : 1x/sem min, 2x/sem optimal, 3x/sem max
### Logique sous-jacente
- TikTok aime la **régularité** > rafale
- Reels pénalise **moins** la sur-publication mais récompense l'équilibre Reels + posts statiques + Stories
- Shorts récompense les comptes qui mixent Shorts + long-form (subs cross-pollination)
## 2. Best times — audiences FR
### Créneaux haute audience FR (Heure de Paris)
- **Lun-Ven 12h00-14h00** : pause déj (peak workforce 25-35)
- **Lun-Ven 18h00-22h00** : tranches étudiants + jeunes actifs
- **Sam 11h00-14h00** : matinées weekend
- **Dim 19h00-23h00** : Sunday scrolls (peak hebdo Gen Z)
### TikTok globaux (Sprout Social, RecurPost 2026)
- Mar-Jeu 14h-18h
- Lun 17h, Mar 10-13h, Mer 16-18h
- **Posting 30-60 min avant peak** = optimal pour batch testing algo
## 3. First-week velocity — mécanique
- Algo TikTok 2026 : **follower-first testing** — nouveaux followers voient en priorité
- 60 premières minutes : viewing intensity = signal clé
- **50+ engagement points sur 300 premières vues** = trigger push large
- Déclencheurs typiques :
	- Comments dans les 5 premières min (répondre tout de suite, c'est OK de fake un thread)
	- Saves + Shares > Likes
	- Watch-through 70%+
- Si vidéo flop sur 24h, l'algo abandonne dans 95% des cas
- Re-upload n'est PAS recommandé (duplicate detection 90%+ accuracy)
### Ritual post-publication
1. Répondre aux 5 premiers commentaires en moins d'1h
2. Pinner un commentaire qui pose une vraie question / engageante
3. Partager dans 1-2 stories Insta + Discord pour boost initial
4. **Ne PAS** demander "like and follow" (algo le voit comme low-quality signal en 2026)
## 4. Cross-platform strategy
### Best practice 2026
- **Tournage en natif** ou rendu sans watermark
- Léger remix par plateforme : caption, audio, 1ère seconde différente
- Décaler les posts de 1-2 heures entre plateformes
- Submagic / OpusClip / Joyspace permettent multi-format avec sub-format ajusté
### TikTok unoriginal content policy (sept 2025+)
- Matériel pris d'autres créateurs = penalty
- "Pseudo-original" (juste text overlay, music swap) = penalty
- Solution : retake 1ère sec, voice différente, edit re-monté
## 5. Faceless AI — milestones
- 100 subs YouTube : 4-8 sem (15-30 vidéos postées)
- 1 000 subs YouTube : ~6 mois (Batch + keyword pattern reproductible)
- YouTube monetization (1k subs + 4k h watch ou 10M Shorts views/90j) : 6-12 mois (Niche résoudue)
- 10k followers TikTok : 1-3 mois si format viral (1 viral suffit)
- 100k followers : 3-9 mois (Cadence + format fixé)
### Tactiques spécifiques faceless
- **Repeatable format** : trouve une template puis fais 30 variantes (économie cognitive viewer)
- **Series mindset** : Episode 1, 2, 3... crée attente
- **Niche keyword pattern** ("Top 5 X", "What if Y", "How [niche] really works") = 10-50 vidéos derrière la même structure
- **Multilingue** : EN + FR + ES = exposition x3 à effort marginal (avec ElevenLabs dubbing)
- **Cross-channel** : 1 chaine YouTube + 1 TikTok jumeau
## 6. Pre-batch strategy
- Batch 14-30 jours d'avance = sustainable
- Studio day 1x/sem (4-6h) : génération visuel + voice + edit pour 5-7 ép
- Studio day 2 : captions + thumb + scheduling
- Use Postiz / Buffer / Later pour scheduler tout
- Engagement live : 30 min après chaque post (calendrier strict)
## 7. Hashtags + SEO
### TikTok
- 3-5 hashtags : 1 broad (1M+) + 2 niche (10k-500k) + 1-2 trending
- Caption avec keyword principal répété (text overlay aussi)
- TikTok Search SEO 2026 : auto-caption + on-screen text indexed
### Reels
- 3-5 hashtags niche
- Description 1-2 lignes avec keyword clé
- Topics tags (nouvel ajout 2025) : forcer la catégorisation
### Shorts
- Title court avec keyword fort
- 2-3 hashtags dans la description
- #Shorts obligatoire en préfixe titre ou description
## 8. KPIs hebdo à monitorer
- **Avg watch time %** : viser 65%+
- **3-second hold rate** : viser 70%+
- **Shares / view** : 0,5%+
- **Save rate** : 1%+
- **Profile visit rate** : 5-10%
- **Bio link CTR** : 2-5%
- **Follower growth velocity** : +5%/sem
