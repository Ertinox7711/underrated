Vision Produit

C'EST QUOI L'OUTIL

Une app web privée, accessible aux élèves après achat de la formation. Chaque étape du workflow de production est automatisée via des APIs.

❌
Sans l'outil
  • → S'inscrire sur ElevenLabs, configurer la voix
  • → S'inscrire sur Midjourney, apprendre les prompts
  • → S'inscrire sur Runway, comprendre les paramètres
  • → Assembler manuellement dans CapCut
  • → 7h pour un élève débutant = abandon garanti
✅
Avec l'outil
  • → Élève saisit : genre, personnages, épisode N
  • → Clic "Générer Script" → script en 30 secondes
  • → Clic "Générer Voix" → fichier audio prêt
  • → Clic "Générer Visuels" → images + clips vidéo
  • → Export ZIP → importe dans CapCut et monte
💡
Clé différenciatrice formation Underrated : les concurrents vendent des méthodes théoriques. Underrated livre un outil qui fait le travail technique. L'élève se concentre sur la créativité et la narration, pas sur la technique.

Flow technique complet
// Élève clique "Générer" → Frontend appelle Backend

POST /api/generate
{
  "serieId": "abc123",        // Config série depuis Firestore
  "episodeNum": 12,
  "contexte": "Bananello éliminé...",
  "commentaires": "Alliance Cerisa+Fraisio"
}

// Backend orchestre les 4 étapes en async :

// 1. Script via OpenAI GPT-4o
const script = await generateScript(contexte, serieConfig);

// 2. Voix via ElevenLabs
// 3. Images via DALL-E / Replicate
// → Ces deux en parallèle (Promise.all)
const [audio, images] = await Promise.all([
  generateVoice(script.voixoff),
  generateImages(script.scenes)  // 5 scènes
]);

// 4. Vidéo via Runway (image → video, séquentiel)
const clips = await generateVideos(images);

// 5. ZIP + upload Firebase Storage
const zipUrl = await packAndUpload({ audio, images, clips, script });

// 6. Retourne lien de téléchargement à l'élève
return { downloadUrl: zipUrl, expiresIn: "24h" };

Sécurité & Accès

AUTH ÉLÈVES

Seuls les acheteurs de la formation ont accès. Couplé à Teachable pour vérification automatique.

Flow d'accès
// 1. Élève achète sur Teachable (99€)
// 2. Teachable Webhook → Backend crée compte Firebase

POST /webhook/teachable-purchase
{
  "email": "eleve@gmail.com",
  "product_id": "underrated-formation",
  "status": "completed"
}

// Backend crée le compte + envoie email avec lien studio
await firebase.auth().createUser({ email, password: generateTempPassword() });
await firestore.set(`users/${uid}`, {
  plan: "formation",
  credits: 30,             // 30 épisodes inclus
  purchasedAt: Date.now()
});
await sendWelcomeEmail(email, studioLink);
⚠️
Important : chaque call API backend vérifie que l'utilisateur Firebase a bien plan: "formation" ET des crédits disponibles. Sans ça, retourne 403. Empêche le partage de compte.
